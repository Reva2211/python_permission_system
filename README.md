# Binary Permission Checker Using Bitwise Operators

Binary Permission Checker is a beginner-friendly Python mini project that demonstrates the practical use of bitwise operators for managing user permissions such as read, write, and execute access.

The program uses binary values to assign and verify permissions similar to real-world operating system permission handling.

## Bitwise Operators Used
- `|` Bitwise OR
- `&` Bitwise AND

Example:
```python
permission |= read
permission & write
