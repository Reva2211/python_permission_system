# Binary Permission Checker Using Bitwise Operators

This project is a beginner-friendly Python mini project that demonstrates the use of bitwise operators for handling user permissions such as read, write, and execute access.

The program assigns binary values to different permissions and checks access using bitwise operations similar to real-world permission systems used in operating systems.

## Features
- Read permission handling
- Write permission handling
- Execute permission checking
- Permission value calculation
- Binary-based access control

## Bitwise Operators Used
- `|` Bitwise OR
- `&` Bitwise AND

## Technologies Used
- Python

## Concepts Covered
- Bitwise Operators
- Conditional Statements
- Variables
- Loops
- Binary Permission Logic

## Permission Values
| Permission | Value |
|------------|-------|
| Read       | 1     |
| Write      | 2     |
| Execute    | 4     |
